# Security Policy

## 🔒 Smart Contract Security

Urutoken V2 is built using OpenZeppelin standards and follows best practices for smart contract security.

## 🛡 Key Protections

* Multi-signature wallet (Safe Wallet)
* Vesting mechanism (3 years lock)
* Verified contract on Etherscan

## ⚠️ Reporting Issues

If you find any security issue, contact the team through official channels.

Do not publicly disclose vulnerabilities.
