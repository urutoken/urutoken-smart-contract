# Urutoken (URU)

Urutoken (URU) is an Ethereum-based utility token designed to support a real-world retail ecosystem and connect traditional commerce with blockchain technology.

## Token Details

- Name: Urutoken
- Symbol: URU
- Network: Ethereum
- Standard: ERC-20
- Total Supply: 1,000,000,000 URU
- Decimals: 18
- Contract Address: 0x7e050631454a46be27A835e24A95DbddeB8Ed03a

## Purpose

Urutoken is intended to support retail integration, customer engagement, and ecosystem services within a broader commerce model.
